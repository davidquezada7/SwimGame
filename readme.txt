Swim Game

Swim Game es un juego multiplayer escrito en Java utilizando libGDX como motor de gráficos.

Características técnicas del juego:

Servidor central para la sincronización de posiciones.
Sincronizado por medio de impulsos y posiciones.
Deteccion de colisiones entre obstaculos, bordes y el jugador.
Sprites Animados y sincronizados.
